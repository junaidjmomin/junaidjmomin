### the short version

I'm a Computer Engineering student who got tired of tutorials that stop right before the interesting part, so most of what I know came from breaking things past the tutorial and figuring out why. In 2025 that habit paid off — **Smart India Hackathon**, first place, built under the kind of deadline pressure that makes you discover features of your own code you didn't know existed.

Right now I split my attention between two things that don't get along:

```
if (problem.needs("someone to just decide and act")) 
    → build an AI agent for it
else if (problem.needs("no one to be in charge"))
    → see what Web3 can do instead
```

I like AI assistants that actually *do* the task instead of describing how you'd do it, dashboards that make messy data feel less messy, and Web3 flows simple enough that my non-technical friends stop asking "wait, so where's the button."
